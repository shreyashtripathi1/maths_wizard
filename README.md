# Maths Wizard

Maths Wizard is a fast-paced local multiplayer math quiz game built in Unity where two players compete against each other on the same keyboard. Players solve math questions as quickly as possible, earn points, and use spells strategically to reduce their opponent’s health.

---

## Gameplay

* Two players play on the same laptop/keyboard.
* Each math question contains 4 options.
* The player who answers correctly first earns points.
* Players can use earned points to cast special spells.
* The goal is to reduce the opponent’s health to zero.

---

## Controls

### Player 1 Controls

```text
A -> Option 1
S -> Option 2
D -> Option 3
F -> Option 4
```

### Player 2 Controls

```text
H -> Option 1
J -> Option 2
K -> Option 3
L -> Option 4
```

---

## Spell System

Players can spend earned points to use spells against their opponent.

| Spell              | Cost     | Effect                       |
| ------------------ | -------- | ---------------------------- |
| Basic Attack Spell | 2 Points | Reduces opponent health by 2 |
| Heavy Attack Spell | 4 Points | Reduces opponent health by 5 |

## Getting Started

### Prerequisites

Make sure you have the following installed:

* Unity Hub
* Unity Editor
* Visual Studio

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/shreyashtripathi1/maths_wizard.git
```

2. Open the project using Unity Hub.

3. Open the main scene inside Unity.

4. Press the **Play** button to start the game.
